2. сетевой протокол SSH настроен:
Установка OpenSSH: sudo apt install openssh-server
Включить автозапуск: sudo systemctl enable --now ssh
Настроить безопасность: sudo nano /etc/ssh/sshd_config
Изменить: PermitRootLogin no, PasswordAuthentication no
Перезапустить SSH: sudo systemctl restart ssh
Проверка: sudo systemctl status ssh, Active: active (running) — сервис работает
Проверить открытый порт (22): sudo ss -tulnp | grep ssh, 
tcp   LISTEN  0  128  *:22  *:*  users:(("sshd",pid=1234,fd=3))
Проверить подключение с локального компьютера: ssh localhost
3. настроен удаленный доступ к активной сессии:
Установить xrdp для RDP-доступа: 1) sudo apt install xrdp
2) sudo systemctl enable --now xrdp
Разрешить RDP в брандмауэре: sudo ufw allow 3389
Выполнена настройка интернет-соединение: 
1. выбран правильный сетевой интерфейс.
2. продемонстрирована проверка сетевого соединения:

Проверить интернет: 1) ping 8.8.8.8
2) ping google.com
