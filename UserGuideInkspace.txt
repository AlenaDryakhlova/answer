1. Введение
1.1. Область применения
Inkscape — это профессиональное средство автоматизации для создания и редактирования векторной графики. Применяется в:
Дизайне логотипов, иконок, инфографики.
Подготовке макетов для печати (визитки, плакаты).
Создании технических чертежей и схем.
1.2. Краткое описание возможностей
Работа с кривыми фигурами, текстом.
Поддержка слоёв, градиентов, прозрачности.
Экспорт в SVG, PNG, PDF, EPS и другие форматы.
1.3. Уровень подготовки пользователя
Базовые навыки работы с графическими редакторами.
Знание основ векторной графики (рекомендуется).
1.4. Перечень документации
Официальная документация: https://inkscape.org/learn/.
Руководства по SVG.

2. Назначение и условия применения
2.1. Виды деятельности
Графический дизайн.
Вёрстка полиграфии.
Создание векторных иллюстраций.
2.2. Условия применения
ОС: Linux, Windows, macOS.
Минимальные требования: 2 ГБ ОЗУ, 1 ГБ места на диске.
Рекомендуемые настройки:
Разрешение экрана не менее 1280×720.
Мышь с колесом прокрутки или графический планшет.

3. Подготовка к работе
3.1. Установка
Linux (Ubuntu/Debian):
sudo apt install inkscape
 
Windows/macOS:
Скачайте установщик с официального сайта.
3.2. Проверка работоспособности
Запустите Inkscape.
Создайте новый документ (Файл → Новый).
Проверьте инструменты (например, нарисуйте прямоугольник).

4. Описание операций
4.1. Создание объекта
Условия: Открытый документ.
Действия:
Выберите инструмент «Прямоугольник» (F4).
Перетащите курсор для создания фигуры.
Ресурсы: CPU 5–10%, ОЗУ ~200 МБ.
4.2. Экспорт в PNG
Файл → Экспортировать.
Укажите разрешение и путь.
Нажмите «Экспорт».

5. Аварийные ситуации
5.1. Inkscape не запускается
Решение:
bash
Copy
Download
inkscape --debug
Проверьте ошибки в терминале.
5.2. Повреждение файла
Восстановите автосохранённую версию в папке:
Linux: ~/.config/inkscape/autosave/
5.3. Несанкционированный доступ
Закройте программу и проверьте систему на вирусы.

6. Рекомендации по освоению
6.1. Контрольный пример
Создайте логотип:
Добавьте текст (инструмент «Текст»).
Примените градиент («Заливка и обводка»).
Экспортируйте в SVG и PNG.
6.2. Полезные ресурсы
Официальные уроки: https://inkscape.org/learn/.
